# Clinical Trial Analysis - SAS

SAS-based clinical demographics summary table (Table 1.1) — Internshala certification project

## What this project does
- Imports clinical trial data from Excel using PROC IMPORT
- Derives patient age from date of birth / visit date
- Applies custom formats for demographic variables
- Computes summary statistics with PROC MEANS and PROC FREQ
- Uses macro variables for reusable, parameterized code
- Outputs a formatted Table 1.1 demographics summary as RTF

## Tools used
- SAS Studio
- PROC IMPORT, PROC MEANS, PROC FREQ, PROC FORMAT
- SAS Macro variables

## Files
- `table1_demographics.sas` — main SAS program
- `clinical_data.xlsx` — sample dataset
- `table1_output.rtf` — final formatted output
- `certificate.pdf` — Internshala certification

## About
Completed as part of the Clinical Trial Analysis & Reporting certification from Internshala.
